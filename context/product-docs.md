# CloudFlow Product Documentation

## Password Reset
Go to login page → click "Forgot Password" → enter your email → check inbox for reset link → link expires in 30 minutes.

## API Access
API keys are found under Settings → Developer → API Keys. Each plan has rate limits: Starter=100 req/day, Pro=1000 req/day, Enterprise=unlimited.

## Team Members
Invite via Settings → Team → Invite Member. Enter email and select role: Viewer, Editor, or Admin.

## Integrations
CloudFlow integrates with Slack, GitHub, Jira, and Zapier. Found under Settings → Integrations.

## Export Data
Go to Settings → Data → Export. Supports CSV and JSON. Enterprise users get PDF export.

## Mobile App
Available on iOS and Android. Download from App Store or Google Play. Same login credentials as web.

## Billing
Billing is managed under Settings → Billing. Upgrade/downgrade anytime. Refunds handled by billing team only.

## Bug Reporting
Report bugs via Settings → Feedback → Report Bug. Include steps to reproduce.
